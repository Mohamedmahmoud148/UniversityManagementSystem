# 04 — API Catalog

> Complete reference for every backend endpoint. For each endpoint: method, route, auth, request, response, errors, frontend usage.

## Base URLs
```
.NET API:   VITE_API_BASE_URL  (e.g. https://api.university.edu)
AI Service: VITE_AI_SERVICE_URL (e.g. https://ai.university.edu)
```

## Authentication (`/api/auth`)

### POST /api/auth/login
```
Auth: None
Request: { username: string, password: string }
Response: { token: string, refreshToken: string, expiresIn: number, user: UserProfileDto }
Errors: 401 Invalid credentials, 400 Validation
Frontend: Login page form submit
Cache: Never cache
```

### POST /api/auth/refresh-token
```
Auth: None
Request: { token: string, refreshToken: string }
Response: { token: string, refreshToken: string, expiresIn: number }
Errors: 401 Invalid/expired refresh token
Frontend: Axios interceptor on 401
Cache: Never cache
```

### POST /api/auth/change-password
```
Auth: Any authenticated
Request: { currentPassword: string, newPassword: string }
Response: 204 No Content
Errors: 400 Weak password, 401 Wrong current password
Frontend: Profile Settings > Security tab
```

### GET /api/auth/me
```
Auth: Any authenticated
Response: { id, role, profileId, fullName, universityEmail, ...role-specific fields }
Frontend: Called on app init to populate user store
Cache: staleTime: 10min, refetchOnWindowFocus: true
```

### POST /api/auth/logout
```
Auth: Any authenticated
Request: { refreshToken: string }
Response: 204 No Content
Frontend: Logout button → clear localStorage → redirect to /auth/login
```

### POST /api/auth/admin/reset-password/{userId}
```
Auth: Admin, SuperAdmin
Response: { temporaryPassword: string }
Frontend: Admin User Management > Reset Password modal
```

## Exams (`/api/exams`)

### POST /api/exams
```
Auth: Doctor, SuperAdmin

  title, description, type: "Quiz|Midterm|Final",
  totalMarks, passingMarks,

  subjectOfferingId,

```

### POST /api/exams/generate-ai
```
Auth: Doctor, SuperAdmin
Request: CreateAiExamRequest {
  subjectOfferingId, numQuestions, difficulty: "Easy|Medium|Hard",

Frontend: Doctor > Exam Management > "Generate with AI" button
Loading: Full-page spinner "AI is generating your exam..."
```

### GET /api/exams/my-exams

## FastAPI AI Endpoints

### POST /api/chat (FastAPI)
```
Base: VITE_AI_SERVICE_URL
Auth: Bearer token (forwarded)
Request: {
  userId, role, message, history[], academicContext, conversationId
}
Response: {
  response: string, intent_executed: string,
  model_used: string, suggestions: string[]
}
Frontend: AI Chat interface main message send
Loading: Streaming typing effect
```

### POST /api/companion/generate-flashcards (FastAPI)
```
Request: { topic, card_count, difficulty }
Response: FlashcardItemDto[]
Frontend: Flashcard generation (called by .NET via companion service)
```

### POST /api/companion/study-plan (FastAPI)
```
Request: AiStudyPlanRequest
Response: AiStudyPlanDto
Frontend: Companion > Study Plan generation
```

### POST /api/companion/quick-prompt (FastAPI)
```
Request: { prompt, lang }
Response: { response: string }
Frontend: Quick AI responses (session feedback, etc.)
```
